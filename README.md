# SecurePass

SecurePass is a client-side password strength checker and analyzer built using HTML, CSS, and JavaScript.

## Features
- Checks password length and entropy
- Detects common patterns and predictable substitutions
- Provides real-time feedback
- Suggests stronger password alternatives

## Security Note
This tool runs entirely on the client side.
No passwords are stored, logged, or transmitted.

## Tech Stack
- HTML
- CSS
- JavaScript

## Future Improvements
- Better entropy calculation
- Testing against leaked password datasets
